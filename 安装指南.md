# Node.js 和 Git 安装指南

## 📦 安装 Node.js

### 步骤 1：下载 Node.js

1. 打开浏览器，访问 Node.js 官网：
   ```
   https://nodejs.org/zh-cn/
   ```

2. 您会看到两个版本：
   - **LTS（长期支持版）** ← 推荐下载这个
   - Current（最新版）

3. 点击 **"下载 LTS"** 按钮（推荐 18.x 或 20.x 版本）

### 步骤 2：安装 Node.js

1. 下载完成后，双击安装包 `node-v20.x.x-x64.msi`

2. 安装向导步骤：
   - 点击 **"Next"**
   - 勾选 **"I accept the terms"**，点击 **"Next"**
   - 安装路径保持默认，点击 **"Next"**
   - 保持所有默认选项，点击 **"Next"**
   - 点击 **"Install"** 开始安装
   - 等待安装完成，点击 **"Finish"**

### 步骤 3：验证 Node.js 安装

1. 按 `Win + R` 键，输入 `powershell`，按回车

2. 在 PowerShell 中输入以下命令：
   ```bash
   node --version
   ```
   应该显示类似：`v20.11.0`

3. 再输入：
   ```bash
   npm --version
   ```
   应该显示类似：`10.2.4`

✅ 如果都显示版本号，说明 Node.js 安装成功！

## 🔧 安装 Git

### 步骤 1：下载 Git

1. 打开浏览器，访问 Git 官网：
   ```
   https://git-scm.com/download/win
   ```

2. 页面会自动开始下载最新版本的 Git for Windows
   - 如果没有自动下载，点击 **"Click here to download manually"**

### 步骤 2：安装 Git

1. 下载完成后，双击安装包 `Git-2.x.x-64-bit.exe`

2. 安装向导步骤（按照以下建议选择）：

   **① 信息页面**
   - 点击 **"Next"**

   **② 选择安装位置**
   - 保持默认路径，点击 **"Next"**

   **③ 选择组件**
   - 保持默认勾选，点击 **"Next"**

   **④ 开始菜单文件夹**
   - 保持默认，点击 **"Next"**

   **⑤ 选择默认编辑器**
   - 选择 **"Use Visual Studio Code as Git's default editor"**（如果安装了VS Code）
   - 或者选择 **"Use Vim"**
   - 点击 **"Next"**

   **⑥ 调整初始分支名称**
   - 选择 **"Override the default branch name for new repositories"**
   - 输入框中填写：`main`
   - 点击 **"Next"**

   **⑦ 调整PATH环境**
   - 选择 **"Git from the command line and also from 3rd-party software"**（推荐）
   - 点击 **"Next"**

   **⑧ 选择SSH可执行文件**
   - 选择 **"Use bundled OpenSSH"**
   - 点击 **"Next"**

   **⑨ 选择HTTPS传输后端**
   - 选择 **"Use the OpenSSL library"**
   - 点击 **"Next"**

   **⑩ 配置行尾转换**
   - 选择 **"Checkout Windows-style, commit Unix-style line endings"**
   - 点击 **"Next"**

   **⑪ 配置终端模拟器**
   - 选择 **"Use Windows' default console window"**
   - 点击 **"Next"**

   **⑫ 选择 git pull 的默认行为**
   - 选择 **"Default (fast-forward or merge)"**
   - 点击 **"Next"**

   **⑬ 选择凭据助手**
   - 选择 **"Git Credential Manager"**
   - 点击 **"Next"**

   **⑭ 配置额外选项**
   - 勾选 **"Enable file system caching"**
   - 点击 **"Next"**

   **⑮ 配置实验选项**
   - 不勾选任何选项
   - 点击 **"Install"** 开始安装

3. 等待安装完成，点击 **"Finish"**

### 步骤 3：验证 Git 安装

1. **关闭之前的 PowerShell 窗口**（重要！需要重新打开才能识别Git）

2. 按 `Win + R` 键，输入 `powershell`，按回车

3. 在新的 PowerShell 中输入：
   ```bash
   git --version
   ```
   应该显示类似：`git version 2.43.0.windows.1`

✅ 如果显示版本号，说明 Git 安装成功！

### 步骤 4：配置 Git（首次使用必须配置）

在 PowerShell 中输入以下命令（替换为您的信息）：

```bash
git config --global user.name "您的姓名"
git config --global user.email "您的邮箱@example.com"
```

例如：
```bash
git config --global user.name "Zhang San"
git config --global user.email "zhangsan@example.com"
```

验证配置：
```bash
git config --global --list
```

## 🐛 常见问题

### Q1: 执行 npm install 报错
**解决方案：**
1. 确保 Node.js 版本 >= 18.x
2. 尝试使用国内镜像：
   ```bash
   npm config set registry https://registry.npmmirror.com
   npm install
   ```

### Q2: 端口 3000 被占用
**解决方案：**
更改端口运行：
```bash
npm run dev -- -p 3001
```
然后访问 `http://localhost:3001`

### Q3: Git 命令找不到
**解决方案：**
1. 确保已关闭旧的 PowerShell 窗口
2. 重新打开新的 PowerShell 窗口
3. 如果还是不行，重启电脑

### Q4: npm 下载速度很慢
**解决方案：**
使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```
